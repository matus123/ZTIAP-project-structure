# Instalacia
- nainstalovat nodejs (https://nodejs.org/en/)
- spustit prikaz v terminali vscode/webstorm/cmd -> npm install

# Spustenie 
- npm run serve:ts
- stranka by mala bezat na http://localhost:1234
